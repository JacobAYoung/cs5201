Detailed Description

template<class T>
class DMatrix< T >

Creates a DMatrix that has multiple methods of doing calculations for that DMatrix.

Author
Jake Young
Precondition
DMatrix object is created
Postcondition
DMatrix can do calculations and is used as a matrix class with only the diagonals able to be accessed within the matrix
